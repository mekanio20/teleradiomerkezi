## Gurnamak

1. Proýekti klonlaň:
```bash
git clone https://github.com/mekanio20/teleradiomerkezi.git
```

2. Proýekt papkasyna giriň:

3. Gerekli paketleri gurnaň:

4. Serweri başlatmak üçin:
